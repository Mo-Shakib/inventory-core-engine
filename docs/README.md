# Website (GitHub Pages)

This folder contains a static marketing website for **Inventory Core Engine**.

## Publish on GitHub Pages

1. Push branch `codex/website-github-pages` to GitHub.
2. In your GitHub repository go to **Settings -> Pages**.
3. Set **Source** to `Deploy from a branch`.
4. Choose branch `codex/website-github-pages` and folder `/docs`.
5. Save. GitHub will publish the site URL.
